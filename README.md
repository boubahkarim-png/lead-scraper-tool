# Free Lead Scraper Tool

A FREE, no-API-key-required lead generation tool for B2B outreach.

## Features

- **NO API KEYS REQUIRED** - Works without any paid services
- **Multi-country support** - France, Switzerland, Belgium, Canada
- **Business email patterns** - Generates realistic business emails
- **CSV export** - Ready for import into any CRM

## Quick Start

```bash
python3.11 lead_generator.py
```

## Output

Generates CSV files in `/root/lead-system/outputs/raw/` with columns:
- company
- email  
- phone
- city
- country
- sector

## Email Patterns Used

Businesses typically use these email patterns:
- `info@company.com`
- `contact@company.com`
- `reservation@company.com`
- `booking@company.com`
- `reception@company.com`
